* 서포터
[1. 가입] → [2. 로그인] → [3. 클라이언트 or 서포터 선택] → [4. 클라이언트 매칭]


1. 가입
  ① 휴대전화번호, 비밀번호, 비밀번호 확인, 이메일, 이름, 생년월일, 성별 입력
  ② 휴대전화 본인인증 통과 후 가입 완료

2. 로그인
case 1: 일반 계정 로그인
  ① 휴대전화번호, 비밀번호 입력 후 '로그인' 버튼 클릭
  ② 로그인에 성공하여 앱 메인 페이지로 이동
case 2: 카카오톡 로그인
  ① '카카오톡으로 로그인' 버튼 클릭
  ② 카카오톡에 등록된 휴대전화번호, 이메일, 이름, 생년월일, 성별 정보 제공에 동의하는 페이지가 출력되고 '동의' 클릭
  ③ 로그인에 성공하여 앱 메인 페이지로 이동

3. 클라이언트 or 서포터 선택
  ① '서비스를 제공하는 서포터입니다.' 버튼 클릭
  ② 클라이언트 매칭 페이지로 이동

4. 클라이언트 매칭
  ① 이달의 누적 수익을 확인
  ② '지원유형'에서 '도보 이동지원'을 선택 후 '수신 시작' 버튼 클릭
  ③ 입력 및 선택한 정보들이 나타나는 '클라이언트 연결 중' 페이지 출력
  ④ 잠시 후 클라이언트가 연결되어 클라이언트의 '이름', '성별', '전화번호'가 출력되는 '연결 성공' 페이지가 나타남
  ⑤ 활동지원을 마치고 '활동지원 종료'를 누르면 '이용해 주셔서 감사합니다.'페이지로 이동되고 내용에 대한 비용을 획득하며, 간단한 별점 평가를 진행함
